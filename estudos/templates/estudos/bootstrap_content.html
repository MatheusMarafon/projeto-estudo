Conceitos Fundamentais

A base de todo layout no Bootstrap. Ele permite criar layouts responsivos de forma fácil. A lógica é:

  • .container ou .container-fluid: O "embrulho" principal para todo o conteúdo. .container tem largura fixa que muda em breakpoints, enquanto .container-fluid tem sempre 100% de largura.
  • .row: Atua como um contêiner para as colunas. As colunas devem ser filhas diretas de uma .row.
  • .col-*-*: As colunas onde seu conteúdo vive. A soma das colunas em uma linha deve ser no máximo 12. Ex: .col-md-8 (ocupa 8 de 12) + .col-md-4 (ocupa 4 de 12).

Uma das partes mais poderosas do Bootstrap. Permitem aplicar estilos diretamente no HTML sem escrever uma linha de CSS.

Espaçamento (Margin e Padding):

A sintaxe é {propriedade}{lado}-{tamanho}. Ex: mt-3 (margin-top de tamanho 3), px-5 (padding no eixo X de tamanho 5).

Eu sou um div com p-3 (padding) e mb-2 (margin-bottom).

Conceitos-Chave
  • Mobile First Os estilos são pensados primeiro para telas pequenas, e depois expandidos para telas maiores.
  • Sistema de 12 Colunas A base de todo layout, permitindo flexibilidade e alinhamento fácil.
  • Breakpoints Responsivos Classes como -sm-, -md-, -lg-, -xl- ativam estilos apenas em determinados tamanhos de tela.
  • Componentes Prontos Reutilize componentes como Navbars, Cards e Modals para acelerar o desenvolvimento.
